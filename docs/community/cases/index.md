# 社区操作案例

这里用于收录贡献者、集成商和用户提交的实际操作流程。适合放在这里的内容包括：

- 在 x86 环境导入某个具体 ONNX 模型。
- 使用公开数据集或演示视频复现某个模型工作流。
- 将 CosmoEdge 接入某个具体下游系统。
- 有助于其他用户避坑的环境、模型或 Pipeline 实践记录。

社区案例应范围明确、材料可复现，并且不替代官方教程。

## 新增案例

请从[案例模板](template.md)开始。每个案例使用一组中英文 Markdown 文件：

```text
docs/en/community/cases/<case-slug>.md
docs/community/cases/<case-slug>.md
```

图片和小型辅助素材放在：

```text
docs/assets/community/<case-slug>/
```

除非维护者明确批准，不要把大模型权重、私有链接或私有数据集提交到仓库。
