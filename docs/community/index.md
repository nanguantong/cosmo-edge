# 社区

社区页面用于收录真实使用场景中的操作案例。这些内容有价值，但通常比官方教程更具体，可能依赖贡献者自己的模型、视频、硬件、Docker 环境或操作路径。

官方教程保持维护者验证过的长期稳定路径。社区案例会做基本清晰度、安全性和可复现性审核，但不等同于面向所有部署环境的官方兼容性承诺。

## 内容分区

| 分区 | 用途 |
| --- | --- |
| [操作案例](cases/index.md) | 社区提交的模型导入、部署实践、集成流程和排障经验。 |
| [案例模板](cases/template.md) | 新增社区案例时应使用的结构。 |

## 贡献规则

- 社区案例不要直接改写官方教程。
- 说明模型、视频和素材来源，并补充许可信息。
- 不要提交大模型权重、私有下载链接、凭据、客户数据或内网细节。
- 截图入库前必须脱敏。
- 案例进入文档站时，中英文页面应保持一致。
