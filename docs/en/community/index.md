# Community

Community pages collect real-world workflows that are useful but too specific for the official tutorials.

Official tutorials describe maintainer-verified, long-lived paths. Community cases may depend on a contributor's model, video, hardware, Docker environment, or operating procedure. They are reviewed for clarity and safety, but they are not compatibility guarantees for every deployment.

| [Case Template](cases/template.md) | Required structure for new community cases. |

## Contribution Rules

- Keep official tutorial changes separate from community case changes.
- Include model, video, and asset sources with license notes.
- Do not commit large model weights, private download links, credentials, customer data, or private network details.
- Sanitize screenshots before adding them to the repository.
- Keep English and Chinese pages aligned when a case is accepted into the docs site.
