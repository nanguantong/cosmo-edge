# 卷一：快速上手

> **阅读时长**：10-15 分钟
> **目标**：从开箱通电到在浏览器中看到 AI 实时检测效果
> **前提**：您已收到预装 CosmoEdge 的边缘计算设备
> **无需额外设备**：使用提供的演示视频，无需摄像头即可完成全部教程操作

## 硬件产品介绍

CosmoEdge基于算能的AI边缘计算设备，采用算能BM1688性能强劲处理器，具备16.0T的强大运算能力，用于IP摄像机视频流的智能化分析。

<!-- 这是一张图片，ocr 内容为：18.98 4RAPTRON 12VCUT DCN ALAFM OUT  RS22 LSE30 LY -->

![](images/img_01.webp)

<!-- 这是一张图片，ocr 内容为：PWR WAN ALM SYS AUDIO SIM TF OUT Z RCY PUR RST -->

![](images/img_02.webp)

核心参数如下：

| 硬件       | 型号规格                                         |
| ---------- | ------------------------------------------------ |
| 处理器     | BM1688                                           |
| 处理器主控 | 8核 ARM A53 @1.6GHz                              |
| 内存       | LPDDR4 8G                                        |
| 存储器     | 64GB                                             |
| AI算力     | 16.0 TOPS                                        |
| 网口       | 10/100/1000Mbps自适应网口 * 2个                  |
| 接口       | USB3.0*2、 TypeC*1 、HDMI*1 、 TF*1 、 SIM*1 |

## 第一步：硬件连接

将设备接入您的局域网并通电。

1. 用网线连接设备的 **ETH 网口** 与您的路由器或交换机

<!-- 这是一张图片，ocr 内容为：HRU USE30 LAN MW 网口 -->

![](images/img_03.webp)

2. **接入电源适配器**

<!-- 这是一张图片，ocr 内容为：HCLID USEBIO LAN WW 电源接口 -->

![](images/img_04.webp)

3. 等待约 **60 秒**，设备启动完成

**预期结果**：

+ 电源指示灯（PWR）：常亮（红色）

<!-- 这是一张图片，ocr 内容为：电源指示灯 PWR WAN ALM SYS XT -->

![](images/img_05.webp)

+ 网络指示灯（WAN）：闪烁（表示网络连接正常）

<!-- 这是一张图片，ocr 内容为：PWR 网络指示灯 WAN ALM SYS US80  DEBUG QUT -->

![](images/img_06.webp)

4. **CosmoEdge静态IP配置**
5. 电脑桌面点击 **开始 **-> **控制面板** ->**网络和Internet **->**网络和共享中心** ->**更改适配器设置** -> **网络连接** -> **以太网** 。（推荐刚开始设备和计算机直连）

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_07.webp)

2. 双击打开以太网配置，找到**Internet 协议版本4（TCP/IPv4）**选项打开。

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_08.webp)

3. 将 IP 地址和子网掩码修改成与CosmoEdge同网段的 IP ，并点击确定。

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_09.webp)

**💡 CosmoEdge默认的网址：WAN：192.168.100.1 **

## 第二步：访问管理平台并修改系统配置

打开浏览器，输入设备的 IP 地址。

1. 推荐使用 **Chrome** 或 **Edge** 浏览器
2. 在地址栏输入：`http://<设备IP地址>`（例如 `http://192.168.100.1`）

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_10.webp)

3. 使用默认账号登录：
   - 用户名：`admin`
   - 密码：`admin`（首次登录后建议修改）

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_11.webp)

成功登录后，您将看到**系统首页（Dashboard）**，显示以下关键信息：

|    指标    |                   说明                   |
| :--------: | :--------------------------------------: |
| CPU 使用率 |                处理器负载                |
| 显存使用率 |          加载任务会占用系统显存          |
| NPU 使用率 |                 NPU负载                 |
| eMMC使用率 |              系统硬盘使用率              |
|   丢包率   | 大于百分之十可能导致系统异常，性能会下降 |

点击**系统管理**->**系统设置**->**时间设置**->**手动校时**->**与计算同步**。（因为和设备直连系统时间不能自动获取）

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_12.webp)

点击**系统管理**->**网络配置**，修改设备的ip地址，设置成和你局域网同一段的。（满足电脑可以访问平台的同时还可以连网）

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_13.webp)

> **⚠️ 如果无法访问管理页面**
>
> + 确认设备与您的电脑在**同一网段**
> + 尝试直接用网线连接设备与电脑（跳过路由器）
> + 检查浏览器是否使用了代理，关闭代理后重试

## 第四步：查看统计结果

AI 检测到的所有告警/统计事件都会自动记录，支持回溯查看。

1. 点击左侧导航栏 **「事件中心」** → **「计数统计」**

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_32.webp)

2. 选择 **查询事件->通道名称->算法服务->点击查询（重复点查询会刷新实时统计结果）**

<!-- 这是一张图片，ocr 内容为： -->

![](images/img_33.webp)

页面说明：离开客流=可视化展示OUT；净流入人数=可视化展示IN。
